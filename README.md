# Companion

This allows you to have multiple bluetooth device connected to same device to listen to same audio

Note:This is simply a GUI wrapper for a small feature of PIPEWIRE, also, it has some extra simple set of features too

## Usage

You can use the appImage directly by running :

```bash
./Companion-x86_64.AppImage
```

in the cloned directory after giving permissions or just double click on the appimage in any GUI file manager

## Installation

To install it into your system, clone this directory, inside the treminal, run these commands :

```bash
cd install
chmod +x install.sh
./install.sh
```

then run it by typing companion on your terminal or, using your app drawer
